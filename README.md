# CUME App

## 1. Visão Geral

Aplicação web para gerenciamento das atividades da CUME, incluindo controle de filas de escalada, documentação de membros e fluxo de autenticação. Os protótipos estão disponíveis no Figma:

- [Figma - CUME Protótipo](https://www.figma.com/design/AwogeUtZLeimrjxvfSwcwB/CUME---Prot%C3%B3tipo?node-id=202-4856&t=e7yBbGKNp4XXIfWh-1)

## 2. Front-end

- **Instalar dependências**
  - `npm install`
- **Rodar em desenvolvimento**
  - `npm run dev`

### 2.1. Documentação do Front-end (Docusaurus)

- **Rodar servidor de documentação**
  - `npm run docs:dev`
  - Disponível em [http://localhost:3000](http://localhost:3000)
- **Gerar/atualizar docs de componentes**
  - `npm run docs:generate`
- **Build de produção**
  - `npm run docs:build`
- **Servir build**
  - `npm run docs:serve`
