# Bem-vindo à Documentação do CUME App

Esta documentação fornece informações completas sobre os componentes, páginas e guias de estilo do projeto CUME App.

## O que você encontrará aqui

- **Componentes**: Documentação completa de todos os componentes React utilizados no projeto, incluindo suas propriedades e exemplos de uso.

- **Páginas**: Descrição das páginas principais e subpáginas da aplicação.

- **Guia de Estilos**: Referência rápida sobre cores, fontes, espaçamentos e padrões de design utilizados no projeto.

## Figma

- As páginas, componentes e estilos seguem o padrão determinado no Figma. Acesse por [este link](https://www.figma.com/design/AwogeUtZLeimrjxvfSwcwB/CUME---Prot%C3%B3tipo?node-id=13-1268&t=UbwmHfKhwC1E81lv-1).
